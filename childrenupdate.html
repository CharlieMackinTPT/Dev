Skillsville Children's Program

The Skillsville Children's Program is a free multimedia educational program for children 5-8.

The program is designed to enhance children's executive function skills and knowledge about diverse career options.

To use the Skillsville Children's Program with your organization, review the materials below.

  • Getting Started

  • Watch the Skillsville Overview video

    Overview of the family program and the curriculum pillars. (13 mins)

  • Confirm eligibility

    To use facilitate the Children's Program you must meet the following requirements:

    • A tablet for each child. Check our approved technology list for best outcomes.
    • Reliable, stable internet, of at least 10 Mbps per user at the hosting location.
    • Ability to unblock/whitelist any websites needed for the program at the hosting location.
    • Confidence facilitating technology, and troubleshooting as needed.
  • Make your Skillsville Platform account

    The Skillsville Platform contains the all the materials needed to run the program. You will need to make an account and create childrenâ€™s accounts for the number of participants you expect to have. Follow the instructions on the website.

    To create your account, go to bit.ly/Skillsville.

  • Implementation

  • Pick a Bundle

    Each bundle focuses on a diffferent career cluster, and contains 5 activities.

    We recommend implementing 1 bundle at a time.

  • Review How-to Materials

    Each activity has an educator resource. Children can not access this resource.

    Select the "How-to" button on the platform next to each activity.

  • Set up Technology

    All items on the Technology Prep Checklist must be complete before implementing each bundle.
